# Ethical Guidelines

## This toolkit is for authorized security testing ONLY.

### You MUST:
- Obtain written permission before any phishing test
- Use only in isolated lab environments for learning
- Never target real users without explicit approval
- Follow all applicable laws in your jurisdiction

### You MUST NOT:
- Conduct unauthorized phishing attacks
- Harvest credentials without consent
- Use techniques against individuals maliciously

Unauthorized phishing is illegal in most jurisdictions including:
- **Australia** — Criminal Code Act 1995
- **USA** — Computer Fraud and Abuse Act (CFAA)
- **UK** — Computer Misuse Act 1990
